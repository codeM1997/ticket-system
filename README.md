# Support Ticket Management

## Prerequisites

- Node.js **v20.19.0** (pinned via `.nvmrc` and `server/package.json` `engines.node`)

```bash
nvm install
nvm use
```

(`nvm use` reads `.nvmrc` automatically once inside repo root.)

## Setup

```bash
cd server
npm install
```

Full setup, migration, seed, and run instructions land in later steps of this spec.
